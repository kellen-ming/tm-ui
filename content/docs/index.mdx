## 当前这套实现覆盖了什么

当前阶段重点在三个方向：

- 搭建独立的 docs 路由与内容层
- 为组件建立单独的文档页面
- 逐步把现有 showcase 能力迁移到文档体系里

- 复用现有的 props controls 和代码生成逻辑
- 接入搜索
- 最后再补 AI 助手
